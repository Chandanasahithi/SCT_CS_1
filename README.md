# SCT_CS_1

## Description
This project is a Python-based Caesar Cipher program developed for SkillCraft Technology Cyber Security Internship Task 1.

It allows users to:
- Encrypt messages
- Decrypt messages
- Use custom shift values

## Features
- Simple and user-friendly
- Supports uppercase and lowercase letters
- Preserves symbols and spaces
- Built using Python

## Example

Input:
HELLO
Shift: 3

Output:
KHOOR
